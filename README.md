# kanacharenji
Sebuah game untuk belajar huruf Jepang dengan cara yang lebih menyenangkan

Anda ingin mengenal huruf Jepang dengan asyik dan mudah? Kana Charenji solusinya !!!
Terdapat 2 mode game :
1. Quiz
2. True or False (unlockable)

Jika anda adalah orang yang awam terhadap huruf-huruf Jepang, anda bisa pergi ke help yang berlogo tanda tanya (?)
Semoga game ini dapat menghibur sekaligus menambah wawasan.
